# Система рекомендаций банка «Стар»

**Название проекта:** Рекомендательная система банка «Стар»

**Стек технологий:**
- Java 17+
- Spring Boot
- H2 Database (файловая, read-only)
- PostgreSQL
- Spring JDBC (JdbcTemplate)
- Spring Data JPA
- Liquibase
- Caffeine Cache
- Telegram Bot API
- Swagger / OpenAPI
- DBeaver (для просмотра БД)

**Краткое описание:**
Сервис рекомендаций банковских продуктов для клиентов банка «Стар». На основе анализа транзакций пользователя система формирует персонализированные предложения продуктов. Поддерживает статические правила рекомендаций (зашиты в код) и динамические правила (создаются через REST API менеджерами). Выдача рекомендаций доступна через REST API и Telegram-бота.
